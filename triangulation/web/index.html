Governess Triangulation Studio

Small geometry, crisp answers.

Use angle-based triangulation in two ways: solve a full triangle from one baseline and two angles, or locate a target from two sensor positions and two bearings. The interface calls the same deterministic Python toolkit that now lives in the repository.

What this solves

Triangle Known side `AB` plus angles `A` and `B`
Target Two sensor coordinates plus two bearing angles
Output Plain-language results and machine-readable JSON
Guardrails Rejects invalid triangles and near-parallel sight lines

Inputs

Switch modes, enter the numbers, and the page will call the matching API route.

Rule of thumb: `A + B` must stay below `180` degrees.
Angles use math coordinates by default: counterclockwise from the positive x-axis.

Results

The cards summarize the answer, and the raw JSON stays visible for debugging or automation.
