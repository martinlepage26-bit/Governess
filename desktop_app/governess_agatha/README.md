# Governess Agatha

Offline desktop paper-builder app for the `Governess` project.

## What is included

- `Governess Agatha.html`: main offline app
- `assets/`: styling, logic, and icon files
- `reference/`: HENRY prompt and workflow reference texts
- `Launch Governess Agatha.bat`: simple Windows launcher
- `launch_governess_agatha.vbs`: silent Windows launcher
- `Install Desktop Shortcut.ps1`: creates a Desktop shortcut with the app icon
- `desktop.ini`: folder icon configuration for Windows Explorer

## Install on Windows Desktop

1. Copy this folder to `C:\Users\<you>\Desktop\Governess Agatha`
2. Run `Install Desktop Shortcut.ps1`
3. Open `Governess Agatha.lnk`

## Notes

- The app is fully local and opens in the default browser.
- The default first tab is `Pipeline`.
- Project state is saved in browser local storage.
